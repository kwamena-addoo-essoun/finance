# Compliance Documents

This directory contains Finsight's internal compliance documentation required under the **FTC Safeguards Rule (16 CFR Part 314, amended 2023)** and related regulations.

| Document | Requirement | Status |
|---|---|---|
| [WISP.md](WISP.md) | §314.4 — Written Information Security Program | ✅ Draft complete — requires signed physical/DocuSign copy |
| [incident-response-plan.md](incident-response-plan.md) | §314.4(h), §314.15 — Incident Response Plan | ✅ Draft complete |

## Required External Actions

- **OpenAI DPA:** Confirm zero-data-retention at platform.openai.com → Settings → Data Controls
- **Hosting provider DPA:** Sign a data processing agreement with your production infrastructure provider
- **WISP signature:** Print or DocuSign `WISP.md` and retain the signed copy
- **Legal review:** Have a lawyer review the Privacy Policy and ToS before onboarding real users

## Review Schedule

Both documents must be reviewed annually (by March 1 each year) and after any material change to the service or a security incident.
